# Finished features
- Chat
- Create & work on companies
- Train
- Publish/view/vote articles
- Congress
- Political parties
- Storage
- Sell/buy items
- Multiple currencies
- Private messages

# Install
- composer install
- npm install
- mv conf.sample.php conf.php
- grunt
- import db.sql in your mysql

# Requirements
- PHP >= 7.0
- MySQL
